# 🚗 Car E-Commerce Platform (MERN Stack)

## 📌 Overview

This is a **full-stack car e-commerce web application** built using the **MERN stack (MongoDB, Express, React, Node.js)** with **Tailwind CSS** for UI styling.

The system includes **user authentication, JWT-based authorization, admin dashboard, cart system, and order management** with protected routes for secure access.

This project demonstrates real-world full-stack development skills including backend APIs, database management, and frontend integration.

---

## ✨ Features

### 👤 User Features

* User Registration & Login
* JWT Authentication
* Browse Car Listings
* Add to Cart Functionality
* Place Orders
* View Order History
* Protected User Routes

---

### 🛠️ Admin Features

* Admin Authentication
* Add / Update / Delete Cars
* Manage Orders
* View All Users
* Admin Protected Dashboard

---

### 🔐 Security Features

* JWT Token-based Authentication
* Password Hashing (bcrypt)
* Protected Routes (Frontend + Backend)
* Role-based Access Control (User / Admin)

---

## ⚙️ System Architecture

* Frontend (React) communicates with Backend APIs
* Backend handles authentication, business logic, and database
* MongoDB stores users, cars, orders
* JWT secures private routes and admin access

---

## 🔄 Core Functional Flow

1. User registers / logs in
2. JWT token generated and stored
3. User browses cars and adds to cart
4. Orders are placed and stored in database
5. Admin manages products and orders via dashboard
